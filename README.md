# OpenOS-V2
### 基于openos开发的系统
增加了许多需要软盘才能安装的东西(如oppm,print3d等)
## 注意: 
1. 使用该系统需要插入英特网卡(无线网卡:你如果需要控制无人机要安, 电脑内置droneclient直接调用, 使用 https://www.mcmod.cn/post/710.html 上提供的firmware.lua(本机不内置)及droneclient.lua)
2. 至于安装教程吗...我也不知道
